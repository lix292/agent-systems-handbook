# 雷达

这个板块是面向领域变化的快速层，不应过早地被强行纳入长期稳定的实验室章节。

## 这里包含什么

- 月度或季度市场更新
- 新框架和协议的关注项
- 值得追踪的发布简短笔记
- 新兴类别和术语变化
- 经过筛选的编辑精选与领域笔记

## 编辑意图

保持条目简短、及时，并且易于更新。这是对贡献者最友好的展示层，也是仓库在不破坏长期稳定结构的前提下保持最新的最快方式。
